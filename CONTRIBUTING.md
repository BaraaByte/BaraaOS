# Contributing to Baraa OS

Thank you for your interest in contributing to **Baraa OS**. This document explains how you can help improve the project, whether by reporting issues, suggesting features, improving documentation, or submitting code changes.

## How to Contribute

### Reporting Issues
Please use the **Issues** section on GitHub to report bugs or problems. When reporting an issue:
- Provide a clear title.
- Describe the steps to reproduce the problem.
- Include expected and actual results.
- Add any relevant logs or screenshots.

### Suggesting Features
If you have ideas for improvements or new features:
1. Create an issue titled *“Feature Request: …”*.
2. Explain the problem you want to solve and why it matters.
3. Describe your proposed solution.

### Submitting Code Contributions
To submit code changes:
1. **Fork** the repository on GitHub.
2. **Clone** your fork locally.
3. Create a new branch for your work:
